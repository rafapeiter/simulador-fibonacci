# ⚙️ Simulador C - Recursão Fibonacci

## 🎯 Problema que o projeto resolve

O conceito de **recursão** é frequentemente um dos maiores obstáculos para estudantes de programação e ciência da computação. Este projeto resolve as seguintes dificuldades:

- **Abstração da Call Stack:** Mostra de forma tangível como o computador empilha as chamadas de função e resolve os problemas do menor (caso base) para o maior.
- **Visualização da Explosão Combinatória:** Ajuda a ilustrar a ineficiência e a complexidade de tempo $O(2^n)$ da abordagem recursiva clássica do Fibonacci. O aluno consegue perceber rapidamente o recálculo de subproblemas idênticos.
- **Feedback Imediato sem Setup:** Permite que iniciantes testem o comportamento do código em C diretamente pelo navegador, sem precisarem configurar compiladores (como GCC ou MinGW) em suas próprias máquinas.
